# **App Name**: DAGM Digital

## Core Features:

- Aspirasi Submission: Users can submit their aspirations and issues through a digital form.
- Aspirasi Summarization: Use an LLM to generate a summary from a longer aspirasi, making it quicker for council members to review.
- Legislative News Feed: Display news and updates related to the council's activities.
- Events Calendar: Showcase upcoming legislative events and meetings.
- Directory: A listing of contact details for council members.

## Style Guidelines:

- Primary color: Emerald green (#059669) to evoke a fresh, official, and trustworthy feeling.
- Background color: Very light desaturated green (#F0FAF5).
- Accent color: Amber (#D97706) to give a touch of gold representing legislative power.
- Body and headline font: 'Plus Jakarta Sans' (sans-serif) for a modern and formal look, as requested.
- Note: currently only Google Fonts are supported.
- Use Phosphor Icons for a modern and consistent look.
- Maintain a clean and structured layout to present information clearly.
- Subtle animations on hover and transitions to improve user engagement.
